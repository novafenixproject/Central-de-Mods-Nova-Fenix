### 🛠️ Instruções de Instalação
1. Baixe os dois arquivos `.jar` acima.
2. Certifique-se de ter o **Fabric API** instalado.
3. Coloque os arquivos na sua pasta `%appdata%/.minecraft/mods`.
4. Inicie o jogo e divirta-se!

> **Dica:** Se você encontrar qualquer erro de "Missing Dependency", verifique se baixou os DOIS módulos.
